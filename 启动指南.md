# 职业规划智能体 - 项目启动指南

## 一、环境准备

- **Python 3.x**（用于前端静态服务 + AI 算法服务）
- **Node.js**（可选，当前前端为静态 HTML，无需 Node）

## 二、启动步骤

### 方式 A：使用脚本启动（推荐）

1. **启动前端**
   - 双击运行：`start-frontend.bat`
   - 或在命令行执行：
     ```bash
     cd 项目根目录
     start-frontend.bat
     ```
   - 前端将运行在 **http://localhost:8888**

2. **启动 AI 服务**
   - 双击运行：`start-ai.bat`
   - 或在命令行执行：
     ```bash
     cd 项目根目录
     start-ai.bat
     ```
   - AI 服务将运行在 **http://127.0.0.1:5001**

3. **访问系统**
   - 在浏览器打开：**http://localhost:8888**
   - 登录账号：`admin`，密码：`123456`（已预填）

---

## 三、服务端口说明

| 服务 | 端口 | 说明 |
|------|------|------|
| 前端 | 8888 | 静态页面，登录、测评、岗位画像等 |
| AI 算法 | 5001 | 职业测评、岗位画像、职业规划报告 |
| Java 后端 | 5000 | 登录、档案等（可选，当前 mock 模式可不用） |

> **提示**：当前 `frontfiles/api.js` 中 `mockMode: true`，登录、档案、推荐等使用本地模拟数据，**无需启动 Java 后端**。

---

## 四、常见问题

1. **端口被占用**
   - 修改 `start-frontend.bat` 中的端口（如改为 3000、8080）
   - 或关闭占用该端口的其他程序

2. **登录失败**
   - 确保使用 `admin` / `123456`（小写）
   - 强制刷新页面（Ctrl+F5）清除缓存

3. **页面打不开**
   - 确认前端服务已启动（终端无报错）
   - 尝试访问 http://127.0.0.1:8888
